## Ping Pong Launcher Main Part

The Main Assembly Part to attach the Shooter Flywheels and Motor to, 6v motors have friction fit, and red motor uses zipties to attach to frame.

## Motor Driving Pinion
![Pinion](img/Pinion.png){: style="max-width:350px; display:block; margin: 0 auto;" }
Attaches to the small 6v motor shaft to allow for driving the flywheel.

## Winch Roller
![Winch](img/Winch.png){: style="max-width:350px; display:block; margin: 0 auto;" }
Winch Roller to pull/release a string for shooter angle control.

## Electronics Board
![Electronics](img/Electronics.png){: style="max-width:350px; display:block; margin: 0 auto;" }
A board to hot glue the electronics required to get the shooter wheels spinning through code.

## Feeder
![Feeder](img/feeder.png){: style="max-width:350px; display:block; margin: 0 auto;" }
Feed balls in/up intake and roll balls out to be sent through flywheel at a set rate of speed.

## Flywheel
![Flywheel](img/Flywheel.png){: style="max-width:350px; display:block; margin: 0 auto;" }
Main Shooting wheel, wrapped in rubber bands for friction and has gear mesh attached for 4:1 ratio to shoot balls.

## Main Assembly Mount
![Mount Shoot](img/Mount_Shoot.png){: style="max-width:350px; display:block; margin: 0 auto;" }
Mount to XRP to attach the main structure and pivot, a 10-32 bolt is used as axle.

## Winch Motor Mount
![Mount Winch](img/Mount_Winch.png){: style="max-width:350px; display:block; margin: 0 auto;" }
Mount to XRP for the winch motor, use zipties to attach motor.
