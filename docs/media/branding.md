## 📝 Team Naming Conventions
Consistency in our name is vital for scouting and community relations.

* **Official Titles:** Use `Team 2064 | The Panther Project` for document titles and social media.
    * *Note:* Use `2064 : The Panther Project` only if the platform does not support the "|" character.
* **Abbreviated:** `FRC Team 2064` or `The Panther Project`.
* **Numerical:** `2064` (Only when FIRST / FRC context is clearly implied).
* **Local Community:** Use `Pomperaug Robotics` when communicating with Middlebury, Southbury, and surrounding towns.

## 🖼️ Logo & Wordmark Usage

### The Primary Logo
The primary logo consists of the **Pomperaug P** with **2064** aligned with the base.
* **Standard Use:** Centered on the first page of major documents (Branding Guidelines, Tech Binders).
* **Vertical Use:** For narrow spaces (backpacks/duffles), use the alternate logo with a scaled "2064" aligned to the edge of the "P".
* **Flair:** An isogrid background may be added for jerseys or high-impact materials.

### Wordmarks
* **Primary:** "Panther Project" in **Red**, above "2064" in **Black**, outlined in **White**.
* **Alternate:** An all-black wordmark is used for grayscale or internal documents.

## 🔴 Color Palette & Typography

### Official Colors
| Element | Hex Code | Use Case |
| :--- | :--- | :--- |
| **Background** | `#FFFFFF` | Standard document background. |
| **Primary Text** | `#000000` | Standard text on light backgrounds. |
| **Isogrid/Heading** | `#414143` | Background flair and accent text. |
| **Primary Accent** | `#CD2030` | High-importance accent text. |
| **Secondary Accent** | `#808285` | Lower-importance accent text. |

### Typography (Roboto Family)
All documents must adhere to the following **Roboto** font specifications:

* **Titles:** 26 pt, Black, Italics, Bold, Title Case.
* **Headings:** 20 pt, Black, Italics, Bold, UPPERCASE.
* **Standard Text:** 12 pt, Regular.
* **Subtitles:** 15 pt, Light, Title Case.
* **Subscripts:** 8 pt, Light.
* **Special Note:** The *Pomperaug Robotics* font is permitted for banners and posters but **never** in official document body text.

---

## 📄 Document Structure Standard
All official Team 2064 documents must contain these structural elements:

1.  **Header:** Alternate wordmark in the top-left corner (Position: X 0.5, Y 0.25).
2.  **Document Title:** Right-aligned in the header using the Heading font structure.
3.  **Footer:** Page count centered in the footer.
4.  **Cover Page:** Must feature the full team name (`Team 2064 | The Panther Project`) in Title font and the primary logo centered.
